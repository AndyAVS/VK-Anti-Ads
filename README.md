# VK-Anti-Ads
## Я не умею писать раширения для браузеров, но написал это "расширение".
Утомила реклама и тупые комменты в ВК? Все эти "заработки за два часа в день на диване" и прочее в весьма годных по контенту пабликах? Комменты от якобы 17-и летних ТП в темах по информационной безопасности или программированию? Расширение ищет элемент с рекламной плашкой или div c комментами в новостной ленте ВК и убирает их из DOM.
### Как установить
* Стянуть к себе исходный код. 
* Проверить его на наличие закладок и майнеров :) 
* Открыть страницу "Расширения" в Хром. 
* Включить режим "для разработчиков". 
* Установить "распакованное расширение" указав каталог с кодом.
