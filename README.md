# VK-Anti-Ads
Утомила реклама в ВК. Все эти "заработки за два часа в день на газоне" и прочее в весьма годных по контенту пабликах. Расширение ищет элемент с рекламной плашкой и убирает родительский блок из DOM. Так как лента новостей не перегружает документ целиком, а подкачивает, использован MutationObserver.
